Clase 5, como subir un proyecto a Github

Pasos en local, como crear un proyecto desde la terminal

  1. abrir la terminal
  2. crear una carpeta del proyecto (mkdir "nombre_de_la_carpeta")
  3. entrar a la carpeta del proyecto (cd "nombre_de_la_carpeta")
  4. creamos un archivo index.html (touch index.html)
  5. iniciar un repositorio git local (git init)

Pasos en Github, como crear un repositorio

  1. entrat a Github
  2. crear un repositorio
  3. copiar la url del repositorio
  4. volver a la terminal
